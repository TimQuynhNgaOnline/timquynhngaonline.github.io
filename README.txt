WEBSITE CHIẾN DỊCH CÙNG TÌM QUỲNH NGA

Cách mở:
1. Giải nén thư mục.
2. Mở file index.html bằng Chrome, Edge, Firefox hoặc Safari.

Cấu trúc:
- index.html: nội dung trang
- styles.css: thiết kế giao diện
- script.js: nút sao chép mẫu tin nhắn
- assets/ban-do-kavkaz.png: bản đồ minh họa

Website không sử dụng thư viện ngoài và có thể đưa lên GitHub Pages, Netlify, Vercel hoặc hosting thông thường.
